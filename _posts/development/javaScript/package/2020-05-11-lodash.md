##### 基础类型操作库 [lodash] 简单处理JS的常用操作
```js
_.isEmpty // 判断对象是否为空
_.includes(collection, value, [fromIndex=0]) // 值是否存在这个集合中 
_.keyBy(collection, [iteratee=_.identity]) //  根据key列表转对象 _.keyBy(array, 'dir') -> {'dir':{}}
```
