## 替换为 Echarts 步骤

```
1. 拷贝技能图谱到新文件
2. 正则替换 \n -> :\n
3. 名称替换 "(.*?)":\s+\[ -> "name":"\$1", "children": [
4. 末级名称替换 "(.*?)":\s+null -> "name":"\$1", "value": 1
5. Ecahrt 图形替换数据展示
```
