## 替换为 Echarts 步骤

```log
1. 拷贝技能图谱到新文件
2. 正则替换 \n -> :\n
3. YML 文件在线转换为 JSON
4. 名称替换          "(.*?)":\s+\[         =====>       "name":"$1", "children": [
5. 末级名称替换       "(.*?)":\s+null         =====>       "name":"$1", "value": 1

```
